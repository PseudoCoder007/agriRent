# Phase 03.3 Validation

## Automated

- `cmd /c node_modules\.bin\tsc.cmd --noEmit`

## Manual

1. Log in as an existing owner account that already has one equipment listing.
2. Delete the listing.
3. Refresh the owner dashboard.
4. Confirm the deleted item no longer appears.
5. Confirm a visible `List equipment` action is still present.
6. Open the deleted equipment URL and confirm it shows the intended not-found state.
7. Toggle dark mode on, log out, and confirm login/signup/forgot-password/reset-password/home remain light/white.

## Exit Criteria

- Owner dashboard only shows active listings.
- Owners can always start a new listing from the dashboard.
- Auth/home pages remain light after logout regardless of prior dark theme state.
