# Plan 04-01: UI/design polish sweep

## Goal

Give the already-built app a coherent, portfolio-ready visual language without changing the product behavior.

## Deliverables

- Consistent typography, spacing, and card treatment across the main routes
- Better hierarchy for landing, auth, browse, detail, dashboard, review, and chat pages
- Designed loading, empty, and error states
- Mobile-friendly refinements for common viewport widths

## Constraints

- No new functionality
- No route structure changes
- No business-rule changes

## Verification

- Desktop and mobile walkthroughs of all core pages
- Typecheck stays green
